1) Quem inventou o sistema de controle de versão Git e por quê?
Linus Torvalds
Ele usava um prorama para manutenção do Linux e decidiu criar o próprio git porque estava insatisfeito com os outros softwares de ersionamento.
2) Quem atualmente é o dono do Github e por quê?
Microsoft que comprou em 2018.
3) Existe outra maneira, além do terminal, de trabalhar com o Github?
Sim. Online através do site do Github, utilizando um aplicativo para Desktop ou pelo vscode.
